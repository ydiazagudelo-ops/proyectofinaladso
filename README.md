# Proyecto Final ADSO - Gestión Integral

Este repositorio contiene la implementación integral del proyecto de formación para el programa ADSO, aplicando un flujo de trabajo profesional basado en las buenas prácticas de DevOps y control de versiones.

## Estado del Proyecto
![CI Status](https://github.com/ydiazagudelo-ops/proyectofinaladso/actions/workflows/main.yml/badge.svg)
*Integración Continua activa y funcional.*

## Descripción
Sistema de gestión desarrollado para la automatización de tareas y la administración de usuarios, integrando un flujo de trabajo que garantiza la trazabilidad, calidad y despliegue continuo de las funcionalidades.

## Tecnologías Implementadas
- **Backend:** Node.js, Express
- **ORM:** Prisma
- **Control de Versiones:** Git & GitHub
- **Automatización (CI/CD):** GitHub Actions
- **Gobernanza:** GitHub Flow, Branch Protection & Semantic Versioning (SemVer)

## Estructura de Trabajo
Este proyecto ha sido desarrollado bajo los lineamientos de la guía integradora:
- **Trazabilidad:** Integración total entre *Issues*, *Pull Requests* y el tablero *Kanban*.
- **Calidad:** Implementación de *Branch Protection* para asegurar que cada *Merge* a la rama `main` pase por revisiones de código y pruebas automatizadas.
- **Versionamiento:** Gestión de lanzamientos mediante *Tags* y *Releases* (v1.0.0).
