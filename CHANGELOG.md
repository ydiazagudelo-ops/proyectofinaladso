# Changelog

Todos los cambios notables en este proyecto se documentarán en este archivo.

## [1.0.0] - 2026-07-06

### Added
- Creación de la estructura base del proyecto y repositorio en GitHub.
- Implementación de la API inicial con Express y Node.js.
- Configuración de rutas básicas para gestión de usuarios.
- Configuración de GitHub Actions para Integración Continua (CI).
- Configuración de Branch Protection en la rama principal.
- Implementación de Semantic Versioning (SemVer) con Tags y Releases.

### Fixed
- Optimización del archivo .gitignore para excluir dependencias innecesarias (node_modules, logs).
- Resolución de conflictos en el historial de ramas durante la integración.

### Changed
- Actualización exhaustiva del README.md con documentación técnica y badges de estado.
- Estandarización de mensajes de commit utilizando Conventional Commits.
